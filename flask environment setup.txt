Get Flask Environment Set up and Running, run following in Ananconda command line

cd "git_repos\flight_price_tracker"

C:\> set FLASK_APP=app.py
C:\> set FLASK_DEBUG=development
C:\> flask run

After flask webserver set up, open another Anaconda command line window and run the following commands to upload and parse json file

cd "git_repos\flight_price_tracker"

curl -X POST -H "Content-type:application/json" -d "@skyscanner_dallas.json" http://127.0.0.1:5000/countries
